## 心理学原理总结

### 1. 投资循环原理（Investment Loop）

```
用户投入时间/数据 → 资产积累 → 沉没成本效应
→ 不想失去已投入 → 愿意付费保留
```

**应用**：
- 收藏机会（数据资产）
- 生成AI报告（知识资产）
- 价格追踪（时间资产）
- 团队协作（社交资产）

### 2. 渐进式价值披露（Progressive Value Disclosure）

```
Day 1: 看到3个卡片（初步价值）
↓
Day 7: 发现可以收藏、追踪（更多价值）
↓
Day 14: 深度使用所有功能（全部价值）
↓
Day 15: 意识到将失去（价值最大化）
```

**应用**：
- 功能逐步解锁
- 价值随使用时间递增
- 到期前才展示全部价值

### 3. 损失厌恶（Loss Aversion）

```
❌ 错误："付费后可以获得更多功能"
✅ 正确："不付费将失去已有的数据和功能"

心理原理：失去$100的痛苦 > 获得$100的快乐
```

**应用**：
- 强调"失去"而非"不能获得"
- 具体化损失（18个收藏、12份报告）
- 展示失去后的不便场景

### 4. 禀赋效应（Endowment Effect）

```
用户拥有某物后，对该物品的价值评估会显著提升

应用：
• 让用户"拥有"收藏机会
• 让用户"拥有"AI报告
• 让用户"拥有"价格追踪数据
→ 失去这些"拥有"时会痛
```

### 5. 社会证明（Social Proof）

```
从众心理：大家都在用，说明有价值

应用：
• "1,234人收藏了这个机会"
• "过去7天收藏量+45%"
• 用户评论和讨论
```
