## 问题诊断

### 当前状态
| 指标 | 现状 | 问题 |
|------|------|------|
| 内容价值 | 纯信息聚合 | 易复制，无壁垒 |

### 用户真实需求
小白卖家不是要"看资讯"，而是要"赚钱"。他们需要：
1. **我能卖什么？** - 选品决策
2. **怎么开始？** - 落地执行
3. **能赚多少？** - 利润预估
4. **风险有多大？** - 风险评估

## 功能设计：让用户留下足迹

### 1. 创业评估问卷 (入口钩子)

**5分钟评估，获得定制化建议**

```
用户填写 → AI分析 → 生成报告
    ↓
留下基础数据：市场、预算、经验、目标
    ↓
【断点】完整报告 + 定制方案 = 注册试用
```

**问卷内容**：
- 你想进入哪个市场？（东南亚/欧美/拉美）
- 你的预算范围？（<1万/1-5万/5万+）
- 你的电商经验？（完全新手/有国内电商经验/有跨境经验）
- 你最担心什么？（选品/物流/运营/资金）
- 你的目标是什么？（副业收入/全职创业/团队扩张）

**立即价值**：
- 免费获得：适合度评分 + 市场建议 + 风险提示
- 完整报告：详细分析 + 定制化学习路径 + 推荐资源

### 2. 创业进度追踪器 (粘性工具)

**让用户看到自己的成长**

```
注册后的第一天：
□ 完成市场评估 (已完成 ✓)
□ 了解目标市场 (进行中...)
□ 选择3个候选产品
□ 完成成本计算
□ 学习平台政策
...
```

**游戏化设计**：
- 完成度进度条
- 每完成一步解锁新功能
- 累计积分兑换奖励
- 成就徽章系统

### 3. 选品工作台 (核心功能)

**从"看资讯"到"做决策"**

```
[产品筛选器]
→ 输入条件：价格区间、目标市场、物流限制
→ AI推荐：10个高机会产品
→ 用户选择：添加到"关注列表"
→ 用户标记：感兴趣/已研究/已排除

【断点】深度分析报告 = 试用
- 供应链分析
- 竞品分析
- 利润预测
- 风险评估
```

### 4. 成本计算器2.0 (实用工具)

**不只是算成本，而是算ROI**

```
输入：产品进价、目标售价、物流成本
↓
AI计算：
- FBA费用估算
- 广告建议预算
- 预期利润率
- 保本销量
- 回本周期

【断点】导出Excel = 试用
【断点】保存多个方案对比 = 付费
```

### 5. 创业档案 (数据沉淀)

**用户的创业数据都在这里**

```
我的档案：
- 已评估的市场：3个
- 已分析的产品：15个
- 已计算的成本方案：5个
- 关注的趋势：8个
- 学习进度：60%

【价值】用户用得越多，离开成本越高
```

## 试用策略：Railway式断点设计

### 3天试用体验设计

```
Day 1: 立即价值
├─ 免费评估报告
├─ 推荐学习资源
└─ 解锁：基础选品工具

Day 2: 深度体验
├─ AI选品分析 (每天3次)
├─ 成本计算器 (完整功能)
└─ 解锁：风险评估工具

Day 3: 临门一脚
├─ 导出功能 (限制)
├─ 定制方案预览
└─ 解锁：订阅优惠 (限时)

Day 4+: 软性限制
├─ AI分析降至每天1次
├─ 无法创建新方案
├─ 已有数据可查看
└─ 引导订阅
```

### 断点设计原则

1. **价值前置** - 先让用户体验到价值
2. **临门一脚** - 在关键时刻设限
3. **已有数据保留** - 用户可以看到但不操作
4. **一次性深度优惠** - 试用结束限时优惠

## 付费理由：从"订阅"到"投资"

### 当前问题
> "为什么我要付费看这些资讯？"

### 新答案
> "我已经投入了3天时间在这里积累数据和方案，付费是为了继续我的创业旅程"

### 价值转移

| 传统订阅 | 新模式 |
|----------|--------|
| 为"内容"付费 | 为"进度"付费 |
| 为"功能"付费 | 为"成果"付费 |
| 为"月费"付费 | 为"投资回报"付费 |

### 具体话术

**试用结束前**：
```
"你已经完成了创业评估 + 分析了5个产品 + 建立了专属档案

继续订阅，你将获得：
- 完整的30天执行计划 (价值：节省2周摸索时间)
- 匹配的供应商资源 (价值：节省3天寻找时间)
- 每周进度检查 (价值：保持创业节奏)
- 导出所有你的数据 (价值：带走你的工作成果)

¥99/月 = ¥3/天
这3元的投资，帮你避免可能的3万元损失"
```
